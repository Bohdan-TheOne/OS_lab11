# Command list

## Server

### As Unauthorised User
- **AUTH** - *Register as new user*
- **LOGIN** - *Login with existing account*
- **QUIT** - *Close connection*

### As Authorised User
- **SEND** - *Send message to all availible user*

### As Moderator only
- **SET** - *Moderate messages*
- **FAULT** - *Get fault list*
